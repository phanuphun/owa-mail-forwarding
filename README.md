### Readme
- i used puppeteer to auto forwarding mail one by one , in my case mail server is disabled auto forwarding mail.
- element selector i use path x instead selector cuase i want to use `locator` in pupeteer 

### Environment
- `MAIL_URL`
- `USR_MAIL`
- `USR_PWD`
- `FORWARD_MAIL`

### Run
- npm run start
- npm run dev
